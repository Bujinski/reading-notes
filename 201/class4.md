# HTML Links, JS Functions, and Intro to CSS Layout

# Learn HTML:

## Q 1. To create a basic link, we wrap text or other content inside what element?

To create a basic link, you wrap text or other content inside the `<a>` (anchor) element in HTML.

## Q 2. The `href` attribute contains what information? 

The `href` attribute in HTML contains the URL (Uniform Resource Locator) or the destination address of a hyperlink.

## Q 3. What are some ways we can ensure links on our pages are accessible to all readers?

Ensure accessible links by using descriptive text, providing alternative text, maintaining logical tab order, avoiding generic link text, using semantic HTML, ensuring distinguishable appearance, and testing with accessibility tools.

> [Creating hyperlinks](https://developer.mozilla.org/en-US/docs/Learn/HTML/Introduction_to_HTML/Creating_hyperlinks)

# CSS Layout:

## Q 1. What is meant by “normal flow”?

"Normal flow" in web design refers to the default layout and positioning of elements on a webpage without any applied CSS positioning.

## Q 2. What are a few differences between `block-level` and `inline` elements?

**Block-level elements:**

- 1. Start on a new line.

- 2. Occupy the full width available.

- 3. Allow setting height and width.

> - Examples: `<div>`, `<p>`, `<h1>`.

**Inline elements:**

- 1. Do not start on a new line. 

- 2. Occupy only as much width as necessary.

- 3. Do not allow setting height and width.

> - Examples: `<span>`, `<a>`, `<strong>`.

## Q 3. ___ positioning is the default for every html element.

Static positioning is the default for every HTML element.

## Q 4. Name a few advantages to using absolute positioning on an element.

1. Precise Placement: Allows precise positioning.

2. Layering: Enables easy layering of elements.

3. No Impact on Layout: Doesn't affect the layout of other elements.

4. Overlapping: Facilitates easy element overlap.

5. Relative to Ancestor: Positioned relative to the nearest positioned ancestor.

6. Escapes Constraints: Can escape parent constraints.

## Q 5. What is a key difference between fixed positioning and absolute positioning?

Absolute positioning is relative to the nearest positioned ancestor, while fixed positioning is relative to the browser window and remains fixed during scrolling.

# Learn JS:

## Q 1. Describe the difference between a function declaration and a function invocation.

Function declaration defines a function, while function invocation calls and executes it.

## Q 2. What is the difference between a parameter and an argument?

Parameter is a function's placeholder, while argument is the actual value passed to it during invocation.
